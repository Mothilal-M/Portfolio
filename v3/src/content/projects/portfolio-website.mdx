The site you're reading. Built as an Astro static site with typed content collections, GSAP-driven motion, and a dark-first (light-capable) design system. A deliberate rewrite of an earlier single-file HTML version.

## Why rebuild

The previous version worked but looked like every other dev portfolio — indigo gradients, glass cards, floating orbs. This rewrite leans into what I actually do (backend engineering) instead of decoration.

## What changed

- Moved from one 2,000-line `index.html` to a typed, component-based Astro project.
- Replaced gradient-heavy dark theme with an editorial ink/paper palette + one signal color.
- Projects became API-style endpoints with `method`, `status`, `problem/approach/outcome`.
- Quotes became structural section openers rather than decoration.
